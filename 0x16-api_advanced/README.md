# 0x16. API advanced
Tasks based on [Reddit API](https://www.reddit.com/dev/api/)'s many endpoints.  There’s a lot of endpoints available, many that don’t require any form of authentication, and there’s tons of information to be parsed out and presented.

## Learning Outcomes
- How to read API documentation to find the endpoints you’re looking for
- How to use an API with pagination
- How to parse JSON results from an API
- How to make a recursive API call
- How to sort a dictionary by value
